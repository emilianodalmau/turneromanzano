# **App Name**: Turnos Manzano

## Core Features:

- User Data Collection: Collect user details (name, last name, DNI, phone, email) through a step-by-step form.
- Appointment Booking: Allow users to select an available date and time for museum visits.
- Authentication: Secure the admin panel using Google Authenticator for two-factor authentication.
- Schedule Configuration: Enable administrators to configure available days and times for appointments.
- Appointment Management: Allow administrators to create, edit, and delete appointments.
- AI Appointment Recommendation: A tool to help admininstrators decide how to allocate available appointment slots and resources based on historical data and predicted demand, utilizing genkit and gemini 2.5 pro.
- Firestore Integration: Utilize Firestore as a real-time database to efficiently manage and synchronize appointment data.

## Style Guidelines:

- Primary color: Dark blue (#22577A) to reflect the 'Ciudad del Agua' (City of Water) identity.
- Background color: Light blue (#73B8E7), a lighter shade from the logo, for a clean and airy feel.
- Accent color: Medium blue (#38A3A5), another shade from the logo, to highlight key interactive elements and calls to action.
- Font pairing: 'Roboto' (sans-serif) for both headlines and body text, offering a modern and accessible design.
- Use icons from Lucide React, depicting apples, landscapes, appointment clocks, and municipal buildings to visually represent services and information, using colors that complement the primary and accent colors.
- A clean, step-by-step layout for the public appointment booking process, guiding users through each stage with clear instructions and progress indicators.
- Subtle animations using Shadcn/UI components, such as fading transitions and loading spinners, to enhance user engagement without being distracting, using animation speeds that feel smooth and responsive.
